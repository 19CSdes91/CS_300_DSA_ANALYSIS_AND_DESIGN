# CS_300_DSA_ANALYSIS_AND_DESIGN
In this course, we learned data structures and basic algorithm design that will help us when creating projects.


What was the problem you were solving in the projects for this course?
I was asked to create a sorting program for the academic advising team at ABC_U. In this program advisors and up load courses, sort courses, and see required prerequisites. 

How did you approach the problem? Consider why data structures are important to understand.
I approached the problem by carefully looking over the different methods on how to store data.I choose to use a Hash table because it is quick to search and retrieve data, which is important for advisors who will nee to beable to quickly and efficiently search through a lot of data without any lag.


How did you overcome any roadblocks you encountered while going through the activities or project?
I over came most of my roadblocks by going really slow, as I went through my project I used the the assignment for module 4 as my starter code. I broke into chunks carefully building my variables, constructors, desctructors, methods, and so on. It was important that not only did I write slow but to test and deg bug after each section so that way I can hopefully save my sanity at the end of my finished product. 

How has your work on this project expanded your approach to designing software and developing programs?
I realized after doing this project that going slow is important, but also knowing my variable names. I know some of them can be long, but truthfully that helped when debugging my code. It has me know what is what, and instead of using something like "temp", I chose to use "dummyNode", so that way I can use what I learned in Zybooks and know where to look when I get stuck.

How has your work on this project evolved the way you write programs that are maintainable, readable, and adaptable?
Again, using variable names to know exactly what is what and where it should go. Also trying to ready it sort of like a sentence out loud, so that when I am writing my code I know that it is making sense and hopefully not forget anything that I am trying to tell the computer.
